
# Rubik Timer

Trang này là một "Rubik Timer" đơn giản để đo thời gian giải (giây + milliseconds).

Chức năng:
- Hiển thị thời gian hiện tại chính xác đến milliseconds.
- Start/Stop bằng nút hoặc phím `Space`.
- Ghi `Lap` bằng nút hoặc phím `L` (khi đang chạy).
- `Reset` để đặt lại thời gian và xóa laps.

Chạy:
1. Mở `d:/html/index.html` trong trình duyệt (hoặc nhấp hai lần tệp trong File Explorer).

Phím tắt:
- `Space`: Start/Stop
- `L`: Lap (khi đang chạy)
